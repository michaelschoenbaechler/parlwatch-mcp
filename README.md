# parlwatch-mcp

ParlWatch is an open-source project designed to provide easy access to the parliamentary data of the Swiss government.

## Development
Install dependencies:
```bash
npm install
```

Type check:
```bash
npm run type-check
```

Build:
```bash
npm run build
```

Run in dev (ts-node):
```bash
npm run dev
```

Run built version:
```bash
npm start
```
